## Deck Menu Actions

<center><img src="/images/docs/deck-popup-menu.png" width="300" /></center>

**Tap "‚ãÆ" on the deck** to open the menu with these options:
- **Browse** (Premium) - View-only carousel (no grading)
- **List Cards** - See all cards in a list
- **Add a new card** - Create a new card quickly at the end
- **Cut** - Move deck to a different folder
- **Rename** - Change the deck name
- **Delete** - Remove deck and all its cards (with confirmation)
- Tap **More** in the deck menu to access:
    - **Sync with Excel/CSV** - Connect to an existing Excel/CSV file
    - **Export to new Excel** - Save deck as .xlsx
    - **Export to new CSV** - Save deck as .csv
    - **Export DB** - Backup deck as .db file (free)
    - **Deck Settings** - Configure auto-sync, forgotten card limits, etc.

## Tips for Organization

- **Use folders by topic** - Group decks by subject, class, or project
- **Keep folder names short** - Easier to navigate on mobile
- **Export regularly** - Back up important decks using "Export DB" from the More menu
- **Search is fast** - Don't over-organize; search works great for finding decks
