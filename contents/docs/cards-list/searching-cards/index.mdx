## Searching Cards

### How to Search

1. **Tap the üîç search icon** (top-right)
2. **Type your query** in the search field
3. **Results filter instantly** as you type
4. **Tap X** to clear search

### Search Features

- **Searches both** term and definition
- **Case-insensitive** - "DNA" matches "dna"
- **Partial matches** - "proto" matches "protocol" and "protein"
- **Real-time** - Updates as you type
- **Highlights matches** (if supported)

### Tips for Searching

- **Find duplicates** - Search for common terms
- **Locate cards to edit** - Search then tap to edit
- **Review specific topics** - Search for keywords
- **Verify imports** - Search for terms you know should exist
